    TokenCrowdfund

    contract TokenCrowdfund

    TokenCrowdfund is a base contract for managing for managing donations, allowing donators to donate tokens to a cause. This contract implements such functionality in its most fundamental form and can be extended to provide additional functionality and/or custom behavior. The external interface represents the basic interface for purchasing tokens, and conform the base architecture for crowdfunds. They are *not* intended to be modified / overridden. The internal interface conforms the extensible and modifiable surface of crowdfunds. Override the methods to add functionality. Consider using 'super' where appropriate to concatenate behavior.

    Source: crowdfund/TokenCrowdfund.sol
